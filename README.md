# 🖋️ Vintage Typewriter

A premium, immersive 2D typewriter experience built with React and Vite. Master your typing with timeless elegance, realistic audio feedback, and professional-grade telemetry.

![Vintage Typewriter Screenshot](typewriter_2d_final.png)

## ✨ Features

- **🎨 Premium Aesthetics**: A meticulously crafted UI featuring a "Remington Deluxe" vintage design, parchment paper textures, and subtle micro-animations.
- **🎹 Immersive Audio**: Integrated Web Audio API system providing realistic mechanical keypresses, carriage returns, and margin bells.
- **🎮 Multiple Modes**:
    - **Free Practice**: Write whatever is on your mind on virtual paper.
    - **Challenge Mode**: Test your speed and accuracy against curated quotes from famous figures.
- **📊 Advanced Telemetry**: Real-time tracking of:
    - **WPM** (Words Per Minute)
    - **Accuracy Percentage**
    - **Character Count**
    - **Elapsed Time**
    - **Error Tracking**
- **💾 Save Your Work**: Export your typed sessions directly to a `.txt` file with a single click.
- **🛠️ Customization**: Toggle sound effects and statistics display to suit your focus.

## 🚀 Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v18 or higher)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/vintage-typewriter.git
   cd vintage-typewriter
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

4. Open your browser and navigate to `http://localhost:5173/`.

## 🛠️ Built With

- **React 19** - UI Framework
- **Vite** - Build Tool/Dev Server
- **Web Audio API** - Procedural Audio Generation
- **CSS3** - Custom Styling and Animations (Glassmorphism & Neumorphism)
